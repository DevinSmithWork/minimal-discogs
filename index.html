About

Minimal Discogs was made by Devin Smith in April of 2020, using the Discogs.com API.



Key Commands

Main Screen

  • TAB to focus input fields and buttons, ENTER / RETURN to trigger.
  • X : ("Execute") Run search.
  • + / - : Increment/Decrement year & run search.
  • M : ("More") Get more releases (if there are more to get).

Release Screen

  • ESCAPE : Close window
  • LEFT / RIGHT ARROW : View Prev/Next release

About Screen

  • ESCAPE : Close window
